# 🤖 RAG-Based AI Chatbot

## 📌 Overview
This project is a Retrieval-Augmented Generation (RAG) based AI chatbot that provides context-aware and accurate responses by combining Large Language Models (LLMs) with external knowledge retrieval.

The chatbot allows users to query documents and receive intelligent, relevant answers in real time.

## 🎯 Problem Statement
Traditional chatbots rely only on pre-trained knowledge, which can lead to outdated or incorrect responses.

This project addresses the problem by integrating external data sources, enabling the chatbot to generate responses based on relevant retrieved context.

## 🚀 Solution
Developed a RAG pipeline using LangChain and Mistral AI to:
- Retrieve relevant information from documents  
- Pass context to the language model  
- Generate accurate, context-aware responses  

An interactive UI is built using Streamlit for real-time user interaction.

## 🛠️ Tech Stack
- **Language:** Python  
- **Frameworks:** LangChain  
- **LLM:** Mistral AI / OpenAI  
- **Embeddings:**  Hugging Face / OpenAI  
- **UI:** Streamlit  

## 🔍 Features
- Context-aware question answering  
- Document-based knowledge retrieval  
- Real-time chatbot interaction  
- Simple and intuitive user interface  
- Scalable architecture for adding more data sources  

## 🧠 How It Works
1. User enters a query  
2. Relevant documents are retrieved using vector similarity search  
3. Retrieved context is passed to the LLM  
4. LLM generates a response based on the context
